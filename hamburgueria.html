Detalhes sobre o projeto

Funcionalidades principais:
Cadastro e Autenticação de Usuários: Sistema completo para registro e login de clientes.
Gerenciamento de Produtos: Interface para adicionar, editar e remover produtos do menu.
Carrinho de Compras: Funcionalidade para adicionar itens ao carrinho e finalizar pedidos.
Histórico de Pedidos: Funcionalidade que exibe os históricos dos últimos pedidos feito
Área Administrativa: Painel para gerenciar pedidos e produtos, facilitando a operação do negócio.

Este projeto reflete meu conhecimento em desenvolvimento full-stack, com foco especial no back-end. Utilizei PHP para construir a lógica, integrando com um banco de dados para armazenar informações de usuários e pedidos. CSS foi empregado para estilização e JavaScript para funcionalidades interativas no front-end. A organização do código segue boas práticas de desenvolvimento, garantindo escalabilidade e manutenção eficiente.

Área do Cliente/Usuario

Funcionalidades

O usuario: É capaz de visualizar todos os produtos cadastrado podendo olhar todos eles, mas caso queira adicionar um ao carrinho usuario precisa esta logado

O cadastro: Caso o usuario não seja cadastrado ele deve criar um cadastro completo, inserindo o seu nome,sobrenome,telefone e endereço de entrega, para assim facilitar o contato na hora da entrega

Adicionar produto ao carrinho: Permite ao usuario adicionar produtos ao carrinho, podendo adicionar ou remover quantos produto quiser do carrinho e podendo tambem colocar qualquer observação que deseja ao produto que quer pedir, como por exemplo pedir para tirar o Molho

Finalizar pedido: O usuario pode finalizar o pedido, confirmando o todos os itens escolhido e depois confirmar o endereço de entrega e então escolher a forma de pagamento

Historico de pedido: O usuario depois de finalizar o pedido ele pode ver no historico todos os pedido feito, podendo olhar o status de entrega e entao assim podendo ficar ligado o seu pedido

Funcionalidades

O dashboard: Aqui o admin consegue visualizar de forma resumida, os ultimos 10 pedidos realizado foi solicidado pelo cliente, consegue ver nos card de cima o total de venda,o ticket medio e quanto pedidos ja teve
