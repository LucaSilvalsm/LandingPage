Detalhes sobre o projeto

Funcionalidades principais:
Cadastro e Autenticação de Usuários: Sistema completo para registro e login de clientes.
Gerenciamento de Filmes: Interface para adicionar, editar e remover filme do menu.
Sistema de Busca: Funcionalidade para buscar o nome do filme.
Separação por Gênero : Funcionalidade que separa os filmes com base no seu gênero
Exibição Maiores Notas: Funcionalidade que separa os filmes com base na sua nota
Este projeto reflete meu conhecimento em desenvolvimento full-stack, com um foco muito especial no back-end. Utilizei PHP para construir a lógica, integrando com um banco de dados para armazenar informações de usuários e pedidos. CSS foi empregado para estilização e. A organização do código segue boas práticas de desenvolvimento, garantindo escalabilidade e manutenção eficiente.

Funcionalidades Principais

Cadastro: Caso o usuario não seja cadastrado ele deve criar um cadastro completo, inserindo o seu nome,sobrenome e email , para assim facilitar a criação da conta para que ele consiga utilizar o site.

Adicionar Filme: Permite ao usuario adicionar filmes, fazendo com que esse filme seja exibido na tela para que todos consigam visualizar .

Adicionar um Review: O usuario pode fazer algum comentario sobre outro filme postado por outro usuario, podendo assim adicionar uma nota e um comentario no filme

Sistema de Nota: Com Base nas notas data pelo usuario, o sistema define a média para o filme, sendo tambem capaz de separar os melhores filmes com base na sua media

Exibição Maiores Notas: Apresenta os filmes com as melhores avaliações.

Buscar Filme: O usuario pode ver todos os filmes adicionados, ver os comentarios e notas de outros usuarios e ver o filme em outro formato

Editar Perfil: O usuario depois de finalizar o pedido ele pode ver no historico todos os pedido feito, podendo olhar o status de entrega e entao assim podendo ficar ligado o seu pedido
